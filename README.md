# Lettuce Print Starter

Fresh Next.js App Router starter for a hybrid print e-commerce + quote website.

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Upload to a new GitHub repo, then import into Vercel. The Root Directory should be the repo root.

## Notes
- Cart is client-side and persists in localStorage.
- Checkout and quote API routes are scaffolded and ready for Stripe / email wiring.
